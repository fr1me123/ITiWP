Обучающее пособие по Bootstrap

1. Меню (Навигация)

Bootstrap предоставляет различные компоненты для навигации: обычное меню (navbar), вкладки (tabs), таблетки (pills), offcanvas меню и выпадающие списки.

1.1 Navbar (Основное меню)

Классы:
  • .navbar — основной контейнер меню
  • .navbar-expand-lg — разворачивание на больших экранах
  • .navbar-light, .bg-light — светлый стиль
  • .nav-item, .nav-link — элементы навигации
  • .dropdown — выпадающее меню

2. Кнопки

Классы:
  • .btn — базовый стиль кнопки
  • .btn-primary, .btn-success и др. — цветовые варианты
  • .btn-outline-* — контурные кнопки
  • .btn-lg, .btn-sm — размеры
  • .d-grid — блочная ширина
  • disabled — отключенная кнопка

3. Карточки

Классы:
  • .card — основной контейнер
  • .card-body, .card-title, .card-text — содержимое
  • .card-img-top — изображение сверху
  • .bg-*, .text-white — стилизация
  • .row.g-0, .col-md-* — горизонтальное расположение
